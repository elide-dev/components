Debugging

Elide speaks the Chrome DevTools Protocol, so you can debug JavaScript, TypeScript, and Python with the tools you already use — Chrome DevTools, VS Code, or any CDP-compatible client.

Note
The debugger attaches over a WebSocket on 127.0.0.1:4200 by default. Pass --inspect=PORT to change it.

Start the inspector

Launch any script with the --inspect flag. Elide pauses on the first line and prints a DevTools URL:

TERMINAL
$ elide --inspect app.ts
Debugger listening on ws://127.0.0.1:4200
For help, see: https://elide.dev/debug
1

Open chrome://inspect and click Configure to add 127.0.0.1:4200.

2

Your target appears under Remote Target. Click inspect to open DevTools.

3

Set breakpoints, step through frames, and inspect scope — across all guest languages.

Tip
Add --inspect-brk to pause before the first statement — useful when the bug is in module top-level code.

Attach from VS Code

Add an attach configuration pointing at the same port:

.vscode/launch.json
{
  "type": "node",
  "request": "attach",
  "name": "Attach to Elide",
  "port": 4200
}
Warning
The inspector exposes full runtime access. Never bind it to 0.0.0.0 or a public interface in production.

node:fs

Partial 31 of 38 methods

The file system module provides an API for interacting with the file system, modeled on POSIX functions. Both callback and promise-based APIs are supported. File watching is not yet implemented.

importconst fs = require("node:fs");
fs.readFile(path[, options], callback) Supported

Asynchronously reads the entire contents of a file. The promises variant returns the contents instead of using a callback.

pathstring | Buffer | URLFilename to read.
optionsObject | stringencoding and flag — defaults to a Buffer.
callbackFunctionCalled with (err, data).
example.ts
const { readFile } = require("node:fs/promises");
const text = await readFile("./config.json", "utf8");
console.log(JSON.parse(text));
fs.writeFileSync(file, data[, options]) Supported

Synchronously writes data to a file, replacing the file if it already exists. data may be a string, Buffer, or typed array.

example.ts
const { writeFileSync } = require("node:fs");
writeFileSync("./out.txt", "hello from elide", "utf8");

Elide Overview

Elide is a polyglot runtime and toolchain packaged as one native binary. It runs JavaScript, TypeScript, and Python; compiles and formats Java and Kotlin; installs npm, PyPI, and Maven dependencies; serves HTTP apps and static sites; and exposes project tools over MCP.

$ curl -sSL --tlsv1.2 elide.sh | bash -s -
